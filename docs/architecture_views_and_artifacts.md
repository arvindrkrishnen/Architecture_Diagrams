# Architecture Views and Companion Artifacts

The skill plans multiple architecture views, not just one diagram.

## Planned views
1. Executive / Business Capability View
2. Logical / Solution Architecture View
3. Technical Services & Infrastructure Detail
4. Integration & Data Flow View

## Optional overlays
- Security & Compliance
- Observability
- Cost / FinOps
- Sustainability

## Companion artifacts
Generated by `generate_companion_artifacts.py`:
- executive_summary.md
- architecture_decisions.md
- component_inventory.json
- capability_map.md

These artifacts make the output easier to review with CIO, architecture governance, and delivery stakeholders.
