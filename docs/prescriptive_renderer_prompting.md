# Prescriptive Renderer Prompting

Renderer prompts are structured with dedicated sections:
- Overall communication goal
- View type and layout strategy
- Component catalog with icon/shape, domain, color family, and purpose
- Business-to-technical mapping
- Relationship and data-flow instructions
- Visual style guide
- Completeness and guardrail checklist

The renderer must use official draw.io icons where possible and fall back to styled rounded rectangles only when an official icon is not available.
